# studious-fiesta

## Descrição

Site pessoal moderno e responsivo, exibindo portfólio e habilidades.

## Estrutura do Projeto

```
studious-fiesta/
├── index.html          # Página principal
├── styles/
│   └── style.css       # Estilos globais
├── scripts/
│   └── script.js       # Funcionalidades JavaScript
├── assets/
│   ├── images/         # Imagens do site
│   └── icons/          # Ícones
├── README.md           # Este arquivo
└── LICENSE             # Licença do projeto
```

## Tecnologias

- HTML5
- CSS3 (Flexbox, Grid, Animações)
- JavaScript (Vanilla)
- Responsivo (Mobile-First)

## Seções do Site

1. **Header** - Logo e navegação
2. **Home** - Apresentação pessoal
3. **Sobre** - Informações e biografia
4. **Projetos** - Portfólio de trabalhos
5. **Contato** - Formulário de contato
6. **Footer** - Links e informações

## Como Usar

1. Clone o repositório
2. Abra `index.html` no navegador
3. Customize o conteúdo com suas informações
